# Getting Started

This page explains the basic setup for Worthify.

## Requirements

- Paper/Spigot compatible server (Minecraft `1.21.x`)
- Java version required by your server build

## Installation

1. Download the latest Worthify jar from Modrinth.
2. Put the jar in your server `plugins/` folder.
3. (Optional) Install **Vault** and an economy plugin to use your main economy.
4. Restart the server.

## First run

After first startup, configuration files are created under:

`plugins/Worthify/`

Edit values, then run:

`/worthify reload`
