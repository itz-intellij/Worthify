## What is Worthify?

Worthify is a Paper plugin that lets players check item values and sell items for money.

### Highlights

- `/sell` to sell hand/all items
- `/worth` to view values
- Sell history GUI
- Optional worth multipliers
- Optional Vault / PlaceholderAPI / ProtocolLib integrations
